# LearnPlaywrightBatch2x

Welcome to the **LearnPlaywrightBatch2x** repository! This project is a structured learning path focused on fundamental JavaScript concepts and rules, setting the foundation for automation testing using Playwright. 

## 📁 Repository Structure

The project is organized into chapters, each covering specific JavaScript topics:

### `chapter_01_Basics`
Contains fundamental JavaScript introductions and basic script execution.
* `01_basic.js`: Introduction to basic JS syntax.
* `02_JS.js`: Additional basic JavaScript concepts.
* `03_JS_Commands.js`: Commonly used JavaScript commands.
* `04_Hotcode.js`: Practical snippets and 'hot code' testing.

### `chapter_02_Javascript_Concepts`
Explores core concepts in JS.
* `05_JS_Basics`: Deep dive into core JavaScript features and mechanics.

### `chapter_03_Identifier_Literals`
Focuses on the rules for creating variables, commenting code, and developer productivity.
* `06_Identifier_Rules.js`: Basic rules for valid JavaScript identifiers.
* `07_Identifier_Rules2.js`: Further examples and edge cases for naming identifiers.
* `08_Comments.js`: Examples of single-line and multi-line comments in JS.
* `VSCode_Shortcuts.md`: A comprehensive guide to essential Visual Studio Code keyboard shortcuts to boost productivity.
* `Identifier_Rules_Examples.js`: Consolidated file with all valid/invalid identifier examples and best practices.

### `chapter_04_Javascript_Concepts`
Explores variables and functions, focusing on variable declarations and scoping.

* `20_const_hoisting.js`: Hoisting and TDZ behavior with `const`.
